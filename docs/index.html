Core Modules

Six layers of the
research pipeline

01 — Signal Engine
13-Rule Scoring System
Transparent scoring across 13 independent rules. ADX gate blocks entries in directionless markets. ATR × 2.5 stop loss adapts to each stock's realized volatility.
13 Rules · ADX Gate · ATR SL
02 — Backtesting
Walk-Forward Engine
Zero look-ahead bias across any date range. Gap-aware exits handle overnight risk correctly. Dynamic position sizing, portfolio cap, and full R-Multiple tracking per trade.
Walk-Forward · Gap-Aware · O(N)
03 — Optimizer
Bayesian Parameter Search
Optuna TPE sampler over 26 signal parameters. 100 trials converge on the parameter set that maximizes Sharpe × WinRate / |MaxDrawdown| on a 7-stock basket.
Optuna · TPE · 26 Params
04 — Scanner
Parallel Watchlist Scan
Scans 263 tickers in parallel and ranks by signal score. Configurable thresholds and CSV export. Runs entirely from the local warehouse — no network calls.
263 Tickers · Parallel · Offline
05 — Sentiment
NLP Analysis Engine
VADER with custom financial keyword boosting. 72-hour exponential time decay. Tail risk detection overrides the aggregate score on single strong negative events.
VADER · Time-Decay · Tail Risk
06 — Warehouse
Offline Data Layer
263 symbols stored as local CSV files (~1.8M rows). Smart incremental sync fetches only new rows. Runtime requires zero internet connection.
263 Symbols · ~1.8M Rows · Offline

Quick Start

Up and running in
four commands

# 1. Clone
git clone https://github.com/youcefbt-dz/MarketLab.git && cd MarketLab

# 2. Install dependencies
pip install -r config/requirements.txt

# 3. Build local warehouse — run once
python core/stock_warehouse.py

# 4. Launch
python main.py

# Optional: tune signal parameters
python analysis/strategy_optimizer.py --apply

The warehouse step downloads 263 symbols as local CSV files. All subsequent runs are fully offline.
