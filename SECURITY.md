# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 3.x | ✅ |
| 2.x | ❌ |
| < 2.0 | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability in MarketLab, please **do not open a public issue**.

Instead, report it privately by emailing:

**boutemedjetyoucef1@gmail.com**

Include in your report:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

You can expect a response within **72 hours**.

---

> MarketLab is an offline research tool. It does not expose any web server, API, or user data by default. Security concerns are most likely related to dependency vulnerabilities or unsafe use of third-party data sources.
